Stroke Guidance Editor

Click a sub-glyph to select it. Arrow sub-glyphs show their centerline as Bezier handles — drag anchor squares to move endpoints, drag round control handles to bend the curve. Double-click on the curve to add a node, Shift+double-click on an anchor to remove a node. Number sub-glyphs (and arrows without centerlines) keep the old translate/rotate mode. Save writes back to the font.
